Chai CSS

What is Chai CSS?

Chai CSS is a simple CSS utility framework written in JavaScript. Instead of writing CSS files, you use class names to style elements.

Just add chai- prefix to utility class names and JavaScript handles the styling.

How it works

Add class names like chai-p-4, chai-bg-red, chai-text-white to your HTML elements.

When the page loads, the JavaScript finds these classes and applies the corresponding styles.

Utilities

Spacing

p-1 to p-5 for padding

m-1 to m-5 for margin

Colors

bg-red, bg-blue, bg-green

text-white, text-black

Layout

flex, grid, block, hidden

w-full, h-screen, etc.
